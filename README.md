## Overview

This is a SvelteKit Auth with Tailwind Forms, Typography, and Aspect Ratio Starter
## Getting started
Create secret auth token:
1. create .env file 
2. generate a secret token and add to .env:
3.        openssl rand -base64 32
4. Intall packages:
5.       npm install
6. Start dev server:
7.       npm run dev
